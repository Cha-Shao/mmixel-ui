Visit [https://ui.elfmc.com/installation](https://ui.elfmc.com/installation) to start with MMixel UI.

## Components

Visit [https://ui.elfmc.com/components/avatar](https://ui.elfmc.com/components/avatar) to check components.

## Quick Start

### 1. Install MMixel UI

```shell copy
# npm
npm i mmixel-ui
```
or

```shell copy
# yarn
yarn add mmixel-ui
```

### 2. Import components and use.

```tsx
import { Button } from "mmixel-ui"

const MyComponent = () => {
  return (
    <Button>Click</Button>
  )
}
```
